<h1> 👨‍💻 Sobre mi </h1>

📊 "Soy Analista de Datos con un trasfondo único en Investigación Judicial, lo que me permite abordar los datos con un nivel de detalle y rigor superior. Me especializo en transformar datos crudos en activos estratégicos, asegurando la integridad de la información en cada etapa del ciclo de vida (ETL, Limpieza y EDA).

🎯 He optimizado la recuperación de cartera mediante segmentación avanzada y automatización de KPIs, y poseo experiencia técnica validada en proyectos de análisis de comportamiento financiero y pruebas de hipótesis estadísticas. Mi enfoque es simple: datos limpios, análisis rigurosos y decisiones de negocio inteligentes.

🛠️ Tech Stack:

_Lenguajes: Python (Pandas, NumPy, SciPy), SQL (PostgreSQL, MySQL).
_Visualización: Power BI, Tableau, Matplotlib, Seaborn.
_Especialidad: Rigor forense, Limpieza de datos y Análisis Estadístico."
